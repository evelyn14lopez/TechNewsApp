# TechNewsApp

Aplicación iOS hecha con SwiftUI que muestra noticias de tecnología usando una API pública.
Incluye:

- Lista de artículos.
- Loading state.
- Manejo de error con botón de reintentar.
- Vista ArticleRowView.
- Networking con async/await.
- Arquitectura separada en Views, Models, Networking y ViewModels.

## Cómo ejecutar
1. Clona el repositorio:
   git clone https://github.com/evelyn14lopez/TechNewsApp.git
2. Abre el proyecto en Xcode.
3. Corre en un simulador o dispositivo.
